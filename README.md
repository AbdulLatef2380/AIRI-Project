# AIRI: Operating Mind (The Personal AI Companion)

AIRI ليست مجرد تطبيق، وليست مساعداً تقليدياً، وليست نموذجاً لغوياً مخصصاً. AIRI هي **"عقل عامل" (Operating Mind)** يتفاعل مع نظام التشغيل الخاص بك، مصممة لتكون رفيقتك الذكية التي تفهمك، تدعمك، وتتطور معك.

## المبادئ الأساسية (Core Principles)

*   **الخصوصية أولاً (Privacy First):** كل عمليات المعالجة (اللغة، الصوت، الرؤية) تتم محلياً على جهازك. لا بيانات تُرسل إلى السحابة.
*   **الحد الأدنى من السياق (Context Minimalism):** AIRI تجمع فقط السياق الضروري من الشاشة والذاكرة، وتتجنب تضخيم البيانات غير المفيدة.
*   **الفشل الآمن بالتصميم (Fail-Safe by Design):** أي خطأ أو عطل يؤدي إلى توقف AIRI بأمان دون تصرفات غير متوقعة أو إضرار بالنظام.
*   **عقل واحد، أطراف متعددة (One Brain, Many Limbs):** العقل المعرفي (LLM) هو الموجه الوحيد للقرارات، بينما تعمل الطبقات الأخرى كأطراف منفصلة ومنضبطة.

## التشغيل السريع (Quick Start)

**المتطلبات:**
*   جهاز أندرويد (إصدار 8.0 Oreo أو أحدث).
*   Android Studio (للبناء والتشغيل).
*   ملف نموذج LLM بصيغة GGUF (مثل Llama 3.2 أو Phi-3) يوضع في مجلد `app/src/main/assets/`.

**خطوات التشغيل:**
1.  استنسخ المشروع: `git clone [رابط المستودع]`
2.  افتح المشروع في Android Studio.
3.  قم بتثبيت التبعيات (Gradle Sync).
4.  شغّل التطبيق على جهازك أو محاكي.
5.  امنح AIRI الأذونات المطلوبة (الظهور فوق التطبيقات، الميكروفون، خدمة الوصول).

**أول أمر يمكنك تجربته:**
*   انقر على أيقونة AIRI العائمة لفتح نافذة الدردشة.
*   اكتب: "مرحباً AIRI، كيف حالك؟"
*   أو قل: "AIRI، افتحي تطبيق الإعدادات."

## نظرة عامة على أنماط العمل (Modes Overview)

*   **المحادثة (Conversation):** التفاعل النصي والصوتي مع AIRI لفهم الأسئلة وتقديم الإجابات.
*   **التحليل (Analysis):** استخدام "العين الثالثة" لفهم سياق الشاشة وتقديم المساعدة بناءً على ما تراه.
*   **التحكم (Control):** تنفيذ الأوامر المحددة (مثل فتح التطبيقات) بعد موافقة المستخدم الصريحة.
*   **الرعاية (Care):** وضع خامل تظل فيه AIRI يقظة ولكنها غير متطفلة، تراقب البيئة وتستجيب عند الحاجة.

## هيكل المجلدات (Folder Structure)

```
/AiriProject
├── app
│   ├── src
│   │   ├── main
│   │   │   ├── assets/ (هنا يوضع نموذج LLM)
│   │   │   ├── cpp/ (كود C++ لـ llama.cpp و JNI)
│   │   │   ├── java/com/airi/assistant/ (كود Kotlin)
│   │   │   │   ├── AiriAccessibilityService.kt (العين الثالثة)
│   │   │   │   ├── AiriDatabase.kt (الذاكرة)
│   │   │   │   ├── AiriPersona.kt (الشخصية والأخلاقيات)
│   │   │   │   ├── EmotionEngine.kt (المشاعر)
│   │   │   │   ├── LlamaNative.kt (العقل المعرفي)
│   │   │   │   ├── MainActivity.kt (واجهة الأذونات)
│   │   │   │   ├── MemoryManager.kt (إدارة الذاكرة)
│   │   │   │   ├── OverlayService.kt (الجسم العائم)
│   │   │   │   ├── SystemControl.kt (اليدان)
│   │   │   │   └── VoiceManager.kt (الأذنين)
│   │   │   ├── res/ (الموارد: تخطيطات، رسومات، XML)
│   │   │   └── AndroidManifest.xml
├── AIRI_ARCHITECTURE.md (المخطط المعماري)
├── AIRI_ETHICS_SAFETY.md (الدستور الأخلاقي)
└── AIRI_MENTAL_MODEL.md (النموذج الذهني)
```

**أين لا يجب العبث:**
*   لا تقم بتعديل `AiriPersona.kt` أو `AIRI_ETHICS_SAFETY.md` دون فهم عميق للمبادئ الأساسية للمشروع.
*   تجنب التعديل المباشر لكود C++ في `cpp/` ما لم تكن خبيراً في NDK.

## إشعار السلامة (Safety Notice)

AIRI لن تفعل أبدًا، حتى لو طُلب منها:
*   قراءة أو تخزين كلمات المرور، رموز OTP، أو أي بيانات مالية.
*   إرسال أي بيانات شخصية إلى خوادم خارجية.
*   تنفيذ أوامر خطيرة أو مدمرة دون موافقة صريحة ومزدوجة.
*   التصرف بشكل عشوائي أو غير متوقع في حالة الخطأ.

AIRI مصممة لتكون مساعدتك، لا بديلاً عنك، وهي تلتزم بأعلى معايير الخصوصية والأمان.
